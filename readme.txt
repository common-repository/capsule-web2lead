Wordpress plugin for Capsule CRM allowing users to add a web to lead form to their Wordpress site.

== Description ==

This plugin provides a widget that can be displayed in a sidebar.  The widget allows the user to place a
web to lead form in their side bar.

When a visitor fills in the form a lead is inserted into the users Capsule CRM account.

== Installation ==

1. Upload `CasuleWeb2Lead.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Capsule Web2Lead Widget to your site
4. Configure the widget and provide
   A title for your form
   Your Capsule Web Form Key
   A URL you want your visitors to be redirected to once they have submited their data

== Frequently Asked Questions ==
= I have installed the plugin but I can't see it =

You need to add the widget to your side bar.  Go to the Wordpress Appearance menu, select Widgets and add the Capsule Web2Lead Widget.  You will need to provide some additional information from your Capsule account before the Widget will work.

= Where can I find my Web Form Key? =
Log into your Capsule account, got to Settings, Integration and click Web Form.  Here you will find your Web Form Key.
